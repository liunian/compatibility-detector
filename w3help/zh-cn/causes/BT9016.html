BT9016: 只有 IE 和 Opera 支持 document.frames

标准参考

无。

问题描述

document.frames 只有 IE Opera 支持。等同于 window.frames。用来取得当前页面内 window 对象的集合。

造成的影响

在 Firefox Chorome Safari 中使用 document.frames 不能获取到 FRAME 元素。

受影响的浏览器

IE6 IE7 IE8 Opera

问题分析

分析以下代码：

<iframe src="xxx.html"></iframe>
<script type="text/javascript">
alert(!!document.frames);
</script>

在各浏览器输出如下：

IE6 IE7 IE8 Opera Firefox Safari Chrome
true false

解决方案

用 window.frames 代替 document.frames。
